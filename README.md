# 妙麗歲數計算機 

這是一個簡單、可愛的網頁工具，用來計算狗狗「妙麗」的年齡，並將其換算為人類年齡。

##  簡介
使用者只需輸入狗狗的出生日期，程式就會根據開發標準自動計算出對應的歲數。

## 計算邏輯
本工具參考了 **American Veterinary Medical Association (AVMA)** 的標準進行換算：
* 狗狗 1 歲時，約等於人類 15 歲。
* 狗狗 2 歲時，約等於人類 24 歲。
* 之後狗狗每增加 1 歲，約等於人類增加 5 歲。

## 使用技術
* HTML5 / CSS3 (純手繪狗狗 UI)
* JavaScript (邏輯運算與 LocalStorage 儲存)
